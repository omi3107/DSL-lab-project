# Label Definition:
✅ Decision
A statement where something is finalized or agreed.
Examples:
“We decided to launch next week.”
“API choice is finalized.”

✅ Task
A statement assigning work to someone.
Examples:
“Rahul will prepare slides.”
“Aisha should contact the client.”

✅ Deadline
A statement mentioning time constraint.
Examples:
“Submit by Monday.”
“Deadline is 15 March.”

✅ Issue
A problem or unresolved discussion.
Examples:
“Pricing is still unclear.”
“We haven’t finalized the feature.”

✅ General
Anything not falling above.
Examples:
“Let’s move to next topic.”
“Good point.”

# Edge Cases
If sentence has task + deadline → label = task
If sentence is vague → label = general
If sentence has “decided” → always decision
If sentence mentions problem → issue
